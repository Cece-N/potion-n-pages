# Potion & Pages 🔮

A mystical tarot card reading web app with animated card flips, minimalist ink-style illustrations, and readings for every card in the full 78-card deck.

## Live Site

[https://cece-n.github.io/potion-n-pages](https://cece-n.github.io/potion-n-pages)

## Features

- **Single Card** or **Past · Present · Future** three-card spread
- Animated card flip reveal
- Minimalist line-art illustrations for all 78 cards
- Full card meanings for Major Arcana and all four suits
- "Do Another Reading" button to reset and draw again
- Starfield background with floating animations

## The Deck

All 78 cards are included:

- **Major Arcana** — The Fool through The World (22 cards)
- **Wands** — Ace through King (14 cards)
- **Cups** — Ace through King (14 cards)
- **Swords** — Ace through King (14 cards)
- **Pentacles** — Ace through King (14 cards)

## How to Use

1. Choose a spread — **Single Card** for a quick reading, or **Past · Present · Future** for a fuller one
2. Click **Draw Cards**
3. Tap any card to flip it and reveal the illustration
4. Read your guidance below the cards
5. Click **Do Another Reading** to start fresh

## Built With

- HTML, CSS, JavaScript (single file, no build tools needed)
- [Tailwind CSS](https://tailwindcss.com) via CDN
- [Cormorant Garamond](https://fonts.google.com/specimen/Cormorant+Garamond) and [Cinzel Decorative](https://fonts.google.com/specimen/Cinzel+Decorative) from Google Fonts
- SVG card illustrations drawn in code

## Editing the Code

Open `index.html` directly in any text editor or via the GitHub editor (click the file → pencil icon). No installation or build process required.
